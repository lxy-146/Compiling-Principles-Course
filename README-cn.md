## 简介
本仓库为包含同济大学2020-2021年编译原理课程的大作业，其功能如下：
1. 词法分析器
2. 语法分析器
3. 语义分析器
4. 中间代码生成器
5. 代码优化器
6. 目标代码生成器

## 项目目录结构
```bash

├─IntermediateCode.cpp  //中间代码生成器
├─IntermediateCode.h

├─parser.h
```
