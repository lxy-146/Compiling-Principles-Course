## Introduction
This repo contain the term project of Tongji University's Compiling Principles Course in 2020-2021.The features are as follows:
1. Lexical Analyzer
2. Parser
3. Semantic Analyzer
4. Intermediate Code Generator
5. Code Optimizer
6. Object Code Generator
